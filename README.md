# Liste & status

Projet minimal – frontend React + backend Express (in-memory) + script CraftOS pour afficher la liste des tâches sur un Advanced Monitor (ComputerCraft / CC: Tweaked).

Quickstart:
1. `npm install`
2. `npm run start`   (start both api on :3000 and vite dev serve
CraftOS usage example (sur l'ordinateur Minecraft):
`monitor.lua http://192.168.0.10:3000/api/items`
